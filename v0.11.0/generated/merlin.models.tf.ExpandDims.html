merlin.models.tf.ExpandDims

class merlin.models.tf.ExpandDims(*args, **kwargs)[source]

Bases: merlin.models.tf.core.tabular.TabularBlock

Expand dims of selected input tensors. Example:

inputs = {
    "cont_feat1": tf.random.uniform((NUM_ROWS,)),
    "cont_feat2": tf.random.uniform((NUM_ROWS,)),
    "multi_hot_categ_feat": tf.random.uniform(
        (NUM_ROWS, 4), minval=1, maxval=100, dtype=tf.int32
    ),
}
expand_dims_op = tr.ExpandDims(expand_dims={"cont_feat2": 0, "multi_hot_categ_feat": 1})
expanded_inputs = expand_dims_op(inputs)
__init__(expand_dims: Union[int, Dict[str, int]] = - 1, **kwargs)[source]

Instantiates the ExpandDims transformation, which allows to expand dims of the input tensors :param expand_dims: Defines which dimensions should be expanded. If an int is provided, all input tensors

will have the same dimension expanded. If a dict is passed, only features matching the dict keys will be expanded, in the dimension specified as the dict values.
