class merlin.models.tf.BinaryClassificationTask(*args, **kwargs)[source]

Prediction task for binary classification.

Parameters

  • target (Union[str, Schema], optional) – The name of the target. If a Schema is provided, the target is inferred from the schema.

  • task_name (str, optional) – The name of the task.

  • task_block (Block, optional) – The block to use for the task.

  • loss (LossType, optional) – The loss to use for the task. Defaults to “binary_crossentropy”.

  • metrics (MetricOrMetrics, optional) – The metrics to use for the task. Defaults to [precision, recall, accuracy & auc].
